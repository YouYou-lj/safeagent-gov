# I4 可验证审计链与任务回放

以规范化事件、版本元数据和哈希链证明 Agent 决策链未被内容、顺序或删除篡改，并在固定依赖下复现任务裁决。

- 当前实现：规范化版本事件、SQLite 原子追加、事件哈希链、HMAC 签名/校验、缺失检测、策略与工具响应快照、角色化脱敏和确定性回放。
- 独立证据：60 条六类篡改 + 20 条 Agent 回放、并发单链、审计故障失败关闭和敏感字段最小化测试。
- 适用边界：证明仓库定义的事件链完整性；不替代外部可信时间戳或硬件根信任。

参见 [可证伪假设](hypothesis.md)、[算法](algorithm.md)、[基线](baselines.md) 和 [证据索引](evidence.md)。
