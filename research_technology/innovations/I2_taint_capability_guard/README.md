# I2 污点传播、能力票据与事务型审批

把“工具是否可调用”和“敏感数据是否可流向目标”联合裁决，并用一次性能力票据与可恢复审批状态机约束 Agent 任务链。

- 当前实现：类型化工具上下文、RBAC/ABAC、最小权限能力票据、污点传播、事务审批、重放保护和任务图异常检测。
- 独立证据：44 条合成 holdout 的 B0—B3 逐样例结果、并发/故障/TOCTOU 测试和六类安全模拟 Server。
- 安全边界：危险动作仅在模拟器或隔离沙箱执行，策略/审计异常时失败关闭。
